Ironbelly

Grin mobile wallet you've deserved

Uses original Rust code and React-Native for the UI. We are working hard to open-source it very soon! Design is heavily inspired by @GBKS work over here.

Current limitations

  • While the app is in beta testing - it operates only in Floonet (Grin testnet)
  • Floonet is very unreliable right now, so your transaction confirmations can take longer (some times way longer) then 1 minute
  • Wallet ecryption is not supported yet, so only empty password is supported now
  • Multi-account is disabled, so only default account is supported for now
  • Please do not close the app during the initial wallet recovery - we are working on making it interruptable, but currently it's not
  • It's a lot of bugs in the app, but you can report them to us - ironbelly@cycle42.com

Public beta testing is coming soon, already here!
